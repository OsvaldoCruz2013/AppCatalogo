## 📲 Como Testar o App Publicado

### Opção 1: Expo Go (Recomendado para Teste Rápido)

1. **Instale o Expo Go** no seu dispositivo móvel:
   - iOS: https://apps.apple.com/app/expo-go/id982107779
   - Android: https://play.google.com/store/apps/details?id=host.exp.exponent

2. **Abra o projeto**:
   - Acesse: https://expo.dev/accounts/osvaldocruz/projects/app-catalogo
   - Clique em "Open in Expo Go"
   - Ou escaneie o QR Code mostrado na página

3. **Testando**:
   - O app vai abrir automaticamente no Expo Go
   - Você verá a lista com 7 produtos
   - Teste a navegação clicando em qualquer produto

### Opção 2: Web Preview

1. Acesse o dashboard: https://expo.dev/accounts/osvaldocruz/projects/app-catalogo
2. Clique em "Open project" ou "Preview"
3. Selecione a opção "Web" para ver o preview no navegador

### Opção 3: Build Nativo (Para Produção)

Para criar um build instalável (APK/IPA):

```bash
# Android (APK)
npx eas build --platform android --profile preview

# iOS (requer Apple Developer Account)
npx eas build --platform ios --profile preview
```
